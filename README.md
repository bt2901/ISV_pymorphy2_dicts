## _Генералны план_
Конечна цєљ jест сгенерировати фаjлы-словники, кторе будут компатибилне с библиотекоjу `pymorphy2` и библиотекоjу `AZ.js` (обедве библиотеки користаjут подобны формат словника, основаны на DAWG).
